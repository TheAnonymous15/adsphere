# AdSphere Moderators API

This directory contains the next-generation content moderation micro-APIs for AdSphere.

## Modules

- `text_moderator.php` – Advanced text moderation endpoint
- `image_moderator.php` – Advanced image moderation endpoint
- `video_moderator.php` – (stub) Video moderation endpoint
- `realtime_moderator.php` – Real-time composite moderation endpoint

All endpoints return JSON and are designed to be called from the ad upload and review flows.
